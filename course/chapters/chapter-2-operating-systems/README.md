# Chapter 2 : Operating Systems & Virtualization

Back to [Course Introduction](../../README.md#-table-of-contents)


<h3>Introduction</h3>

<p>In this Chapter, we will cover the fundamentals around servers and what role they play in I.T. Servers make up the entire cloud and it's what's responsible for all compute.</p>

<p>We'll cover the different types of servers, technologies used to create and provision servers, as well as the skills and basic fundamental knowledge required to manage servers.</p>

<p>This includes the Operating system running on the server, the role of the Kernel and the different type of operating system resources that we will need to know about.</p>

<p>What your will learn:</p>

<ul>
  <li> What servers are</li>
  <li> Server technology & Virtualization</li>
  <li> Operating System functions & resources</li>
  <li> Creating our first virtual server</li>
  <li> Introduction to Linux</li>
  <li> Introduction to Command-line</li>
  <li> Introduction to scripting </li>
</u>
